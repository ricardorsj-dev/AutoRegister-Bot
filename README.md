# 🤖 AutoRegister BOT

Bot de automação desenvolvido em Python utilizando PyAutoGUI para cadastro automático de produtos em sistemas web.

## 🚀 Funcionalidades

- Login automático
- Leitura de planilhas CSV
- Cadastro automatizado
- Navegação automática
- Preenchimento inteligente de formulários

## 🛠 Tecnologias

- Python
- PyAutoGUI
- Pandas

## ▶ Como executar

pip install -r requirements.txt

python src/bot.py
